# e-plantShopping

Tienda online de plantas con carrito de compras y catálogo dinámico.

## Características

- Catálogo de productos
- Carrito de compras
- Interfaz responsiva

## Instalación

``bash
git clone https://github.com/sjhallo07/e-plantShopping.git
``

Abre el archivo index.html en tu navegador.
